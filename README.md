## 🎯 О проекте
**Vulpixel** — это набор профессиональных инструментов для быстрой обработки изображений прямо в браузере. Главная фишка проекта — **100% приватность**. Все операции выполняются локально на устройстве пользователя, файлы никуда не загружаются и не покидают браузер.

---

## ⚡ Ключевые преимущества
* **Абсолютная приватность** — обработка происходит на стороне клиента.
* **Высокая скорость** — мгновенный результат без очередей и ожидания загрузки на сервер.
* **Доступность** — без лимитов, скрытых подписок, рекламы и создания аккаунтов.

---

## 🛠 Функционал и инструменты

### 🚀 Доступно сейчас
* **⚡ Сжатие изображений**
    * Уменьшение веса файлов без видимой потери качества.
    * Пакетная (массовая) обработка нескольких картинок сразу.
    * Ручной выбор выходного формата (`WebP`, `JPEG`, `PNG`) и настройка максимальной ширины.
* **✂ Интерактивная обрезка**
    * Кадрирование под любые нестандартные пропорции.
    * Готовые пресеты для популярных форматов (`16:9`, `1:1`, `Story`, `A4`).

### ⏳ В разработке (Roadmap)
* **🖼 Конструктор коллажей** — создание сравнительных изображений «до/после» с кастомными разделителями, подписями и выбором шрифтов.
* **🔄 Пакетный конвертер** — быстрое перекодирование графических файлов между современными форматами (включая `WebP` и `AVIF`).

---

## 📋 Технологический стек & Архитектура
* **Тип приложения:** Client-side Web App (SPA)
* **Безопасность:** Локальное выполнение операций (Local Processing).
* **Поддерживаемые форматы на входе/выходе:** WebP, JPEG, PNG, AVIF.
